# adventofcode-2022

These are solutions for the [Advent of Code 2022](https://adventofcode.com/2022).

## Prerequisites

* Ruby 3.0 or later
* Rake (Build tool)
* RuboCop (Static analysis)

### Installing prerequisites

```sh
$ gem install rake rubocop
```

## How to run

Run the most recently edited source file:

```sh
$ rake
```
